Aplikasi yang saya buat adalah Presensi Online, dibuat menggunakan Next.js.
Tujuannya adalah untuk mempermudah proses presensi harian siswa secara digital.

Aplikasi ini terdeiri dari beberapa halaman, yaitu :

1. Login Page – tempat pengguna masuk ke sistem.
2. Dashboard Page – menampilkan sapaan, tanggal, dan tombol presensi.
3. Riwayat Page – menampilkan data kehadiran mingguan.

Alur penggunaannya:

masuk ke dalam folder koding aplikais, dan jalankan menggunakan command prompt dan ketik npm run dev setelah itu akan muncul http://localhost:3000 pada bawah command tadi, copy dan masukan link ke dalam chrome atau browser, pada http://localhost:3000, tambahkan /login pada link sebelumnya dan akan muncul halaman login.

Pengguna memasukan username dan passwordnya --> masuk ke halaman dashboard dan klik presensi agar presensi tersimpan --> bisa klik ke bagian riwayat presensi unutk melihat presensi-presensi sebelumnya atau bisa klik recap bulanan untuk melihat statistik masuk atau tidak masuk dalam satu bulan
